> 웹을 산출물로 사용하면 완전히 독점적인 지위를 가질 수 있다!

* 입학처
* 취업처 - 자기분석, 직무분석, 취업스킬
* 창업
* 교직원 교육

* 자기소개서
* 직무분석
* PPT
